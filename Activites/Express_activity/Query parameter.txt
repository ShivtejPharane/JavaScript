2.Detailed Study About Query Parameters
-->
1.What are Query Parameters?
Query parameters are values passed in the URL to send extra information to the server.
They come after ? in a URL.
Syntax:
URL?key=value
Example:
/products?category=mobile
Here:
category → key (parameter name)
mobile → value

2.Why use Query Parameters?
Used to send data without changing route.
Instead of:
/products/mobile
/products/laptop
/products/shoes
Use:
/products?category=mobile
/products?category=laptop
/products?category=shoes
Same route, different data.

3.Structure of Query Parameters
1.Single Query Parameter
/search?name=John
2.Multiple Query Parameters
Use &
/products?category=mobile&brand=samsung&price=30000
Meaning:
category = mobile
brand = samsung
price = 30000

4.Access Query Parameters in Express
Using:
req.query
Example:
app.get('/products',(req,res)=>{
console.log(req.query);
res.send(req.query);
});
URL:
/products?category=mobile&brand=apple
Output:
{
category:"mobile",
brand:"apple"
}
5.Access Specific Query Value
app.get('/search',(req,res)=>{
let keyword=req.query.keyword;
res.send(keyword);
});
URL:
/search?keyword=laptop
Output:
laptop

6.Query Parameters vs Route Parameters
Route Parameter
/user/101
req.params.id
Query Parameter
/user?id=101
req.query.id

7.Difference:
Route Param	                          Query Param
Part of URL path	               Extra data in URL
Used for specific resource	    Used for filters/search
req.params	                         req.query

3. 3-4 Examples of Query Parameters
-->
Example 1 Search
/search?keyword=iphone
Code:
app.get('/search',(req,res)=>{
res.send(req.query.keyword);
});
Output:
iphone

Example 2 Filter Products
/products?category=shoes
Filter only shoes.

Example 3 Sorting
/products?sort=price
Sort by price.
Another:
/products?sort=name
Sort by name.

Example 4 Pagination
/products?page=2
Shows second page.
Multiple:
/products?page=2&limit=10
Meaning:
page 2
10 products per page

Example 5 Multiple Filters
/products?brand=nike&size=8&color=black
Filter by:
brand
size
color

4.Real-Time Use of Query Parameters
1. Google Search
When you search:
google.com/search?q=nodejs
Query parameter:
q=nodejs
q = search query.

2 .Amazon Filters
Example:
amazon.com/products?brand=apple&price=50000
Used for:
Filters
Price range
Brand

3 .Pagination in Websites
/users?page=3
Shows page 3 users.
Very common in APIs.
Example API:
/api/users?page=1&limit=20
4 Sorting Products
/products?sort=lowtohigh
or
/products?sort=rating

5. Login/Search APIs
Example:
/api/search?city=Pune
Returns Pune data.

6. Weather APIs
Example:
/weather?city=London
City sent as query parameter.
